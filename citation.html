EasyCitation

  1. Product Design
  2. UX Design
Context

"Easy Citation" is originally my submission at Adobe Jam Contest 2019. The topic is to create a mobile app that encourage academic integrity. Within only 3 hours, My teammate and I came to the idea of creating a new app to generate citations.

My role

After the Adobe Jam contest, I decided to revamp the entire project, from Product design, UX research to UI visual design.

Type &
Duration

Website design, Chrome extension and Mobile app

Duration: 1st round: 4 hours.
2nd round: 3 months

Problem Statement

Many students do not know how to write citations/references following a specific style like APA, MLA or Chicago. Even if they know how to, they do not store their references in one place, which makes it very hard to trace back references for further use.
There are some citation websites, such as citationmachine.net. However, they display too much ads and do not have browser extension and mobile app yet. The current websites to generate citation is not good enough.

Empathize with Users

Persona
Persona 1: Rahul, college student

1-on-1 Interview
I prepared a list of 10 questions to interview 6 participants face to face about their sharing academic integrity issues.

    Concrete, innocent questions that are easy to answer:

  • What kind of information (e.g. photo, text) that you put citation to?
  • What is your pain point?

    • Questions that cover the main purpose

  • What is the thing you hate the most about creating citation?
  • Are you familiar with different citation styles? Which style you hear about? Which style you actually know how to create it?
  • Which tool you use to tackle plagiarism?
  • Who and where you gonna seek help? Why?
  • Which device/tool you use when you want to learn more about plagiarism? Which source/site?
  • If there is an app/ website/ chrome extension about plagiarism, what do you expect?
  • What can make you try a new-launched citation app?

    • Round off and thanks:

  • Do you want to add anything that we have not included?

Competitor Analysis

Grammarly is not a direct competitor but they have a good product design.

Pros

  • Simply user flow; AI supported with machine learning
  • Support website, browser extension, mobile app

Cons

  • The mobile app version does not work seamlessly on iPhone.
  • The transition from English keyboard to Grammarly keyboard is not smooth.

There has been no big citation generating in the market yet. CitationMachine.net is free but has too many ads. Turnitin pricing is expensive and only available in web version, not browser extension or mobile app.

Define problem

Although there seems to be a number of competitors, none of them can deliver a seamless user flow to create citations. Additionally, there is no browser extension or mobile app to create citation yet.
“Easy Citation” is whole complete package - website, mobile app, browser extension - that help users generating citations with ease.
