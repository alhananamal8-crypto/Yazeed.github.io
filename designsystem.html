Why we need a design system?
As one of the first in-house UX designers, my colleague and I had to build it from scratch. No design files, no previous design team, even the founder didn’t see why we need a design system. I wanted to start building a UI kit right away and tried to answer why with my colleague. Not only it would make my job easier, but how would it benefit the business and users?

With the design system, we can replicate designs quickly by utilizing premade UI components and elements. Teams can continue to use the same elements over and over, and thus risking unintended inconsistency. At that time we only had 2 designers on the team, how it would be like if we grow and have more designers joining us?

The design system is there to benefit the users. Each element is thoroughly researched to make sure it passes contrast test, accessibility, and matches the target user audience.
Color is a tricky part, especially when you have to work with hot color like red, orange or strong tones like purple. Unfortunately, you can’t choose your company color. If the founder likes red for the logo, you’ll have no other way but to go ahead with red design system. However, a balancing secondary colour is a good blend. For example, red goes well with dark indigo. To check which colour goes well with what, I checked out Materio.

ColorBox is a cool tool to define all the shades you need for each color. This site also help you check if the colour passes accessibility standard. Every color between 0–50 is accessible (4.5:1) on black, and every color 60–100 is accessible (4.5:1) on white. I made sure all of the colors passed the accesible standards.

If you are considering using gradient, uiGradients has awesome gradient samples. The best gradients are created by nature. Follow what nature has created and you’ll get great gradients.

Competitive audit
The founder wanted to build a few products: a web-based dashboard, a website, a mobile application. How about creating a Design System for a complex and growing product variety? I didn’t know the answer so I started this project with some heavy research. I read medium articles, studied the guidelines laid out by Google, Facebook, Apple, Airbnb, and Atlassian, and searched for other designers who faced a similar challenge.
